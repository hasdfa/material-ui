# NoSsr API

The NoSsr component API documentation.

## Import

```
import NoSsr from '@mui/material/NoSsr';
// or
import { NoSsr } from '@mui/material';
```

## Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `node` | - | - |
| `defer` | `bool` | `false` | - |
| `fallback` | `node` | `null` | - |

:::info
The `ref` is forwarded to the root element.
